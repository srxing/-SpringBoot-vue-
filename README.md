# 智慧社区物业管理平台 — 系统说明书

---

## 一、系统概述

智慧社区物业管理平台是一个基于 **Spring Boot + Vue 3** 的前后端分离系统，面向小区物业管理人员和业主，提供数字化、智能化的物业管理服务。

---

## 三、系统架构

```
┌─────────────────────────────────────────────────┐
│                   前端 (Vite + Vue 3)            │
│  管理后台: http://localhost:3000                 │
│  业主端:   http://localhost:3000/owner           │
│  登录页:   http://localhost:3000/login           │
├─────────────────────────────────────────────────┤
│                  API 代理 (Vite Proxy)            │
├─────────────────────────────────────────────────┤
│                   后端 (Spring Boot)             │
│              http://localhost:8080               │
├─────────────────────────────────────────────────┤
│               MySQL 数据库 (3306)                │
│                property_management               │
└─────────────────────────────────────────────────┘
```

### 技术栈

| 层 | 技术 |
|----|------|
| 前端框架 | Vue 3 + Pinia + Vue Router |
| UI 库 | Element Plus |
| 图表 | ECharts + vue-echarts |
| 构建工具 | Vite 5 |
| 后端框架 | Spring Boot 2.7 + Spring Security |
| ORM | MyBatis-Plus |
| 数据库 | MySQL 8 |
| 认证 | JWT（Bearer Token） |

## 四、启动方式

### 前置依赖

- **Node.js** 20.19+ 或 22.12+
- **JDK** 1.8+
- **MySQL** 8.0+
- **Maven** 3.6+（或使用 IDE 内置）

### 第 1 步：初始化数据库

执行 [property_management.sql](sql/property_management.sql)

### 第 2 步：启动后端

```powershell
cd backend
mvn spring-boot:run
```

或直接在 IDE（IntelliJ IDEA）中运行 `PropertyApplication.java`

后端启动在 **http://localhost:8080**

### 第 3 步：启动前端

```powershell
cd frontend
npm install
npm run dev
```

前端启动在 **http://localhost:3000**

## 八、项目目录结构

```
├── sql/                        # 数据库脚本
│   ├── ddl.sql                 # 建表语句（全部表结构）
│   └── dml.sql                 # 初始数据 + 测试数据
├── backend/                    # 后端（Spring Boot）
│   └── src/main/java/com/property/
│       ├── common/             # 通用工具（JWT、Security、异常处理）
│       ├── modules/
│       │   ├── basic/          # 基础资料（楼盘/楼栋/单元/房屋/业主）
│       │   ├── finance/        # 收费管理（账单/缴费）
│       │   ├── parking/        # 停车管理（车位/车辆）
│       │   ├── report/         # 报表中心
│       │   ├── service/        # 物业服务（公告/报修/投诉/访客/活动）
│       │   └── system/         # 系统管理（用户/角色/菜单/部门/字典）
│       └── PropertyApplication.java
├── frontend/                   # 前端（Vue 3）
│   └── src/
│       ├── layouts/
│       │   ├── OwnerLayout.vue # 业主端布局（底部TabBar）
│       ├── views/
│       │   ├── admin/          # 管理后台页面
│       │   ├── owner/          # 业主端页面（home/notice/repair/complaint/bill/payment/activity）
│       │   ├── login/          # 登录页
│       │   └── layout/         # 管理后台布局（AdminLayout/Navbar/Sidebar）
│       ├── router/             # 路由定义 + 角色守卫
│       ├── store/              # Pinia 状态管理
│       └── utils/              # Axios 请求工具
```
